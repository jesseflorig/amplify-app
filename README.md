# [WIP] Amplify Starter App

A quickstart Amplify app with custom authentication workflow

## Setup

Install dependencies:
```sh
yarn
```

Initialize Amplify:
```sh
amplify init
```

Deploy Authentication resources:
```sh
amplify add auth
amplify push
```

Start the dev server:
```sh
yarn start
```
